##   
3. Как запускать из командной строки Windows

**Шаг 1: Открой командную строку (cmd)**

- Нажми `Win + R`
    
- Введи `cmd`
    
- Нажми Enter
    

**Шаг 2: Перейди в папку проекта**

bash

cd C:\Users\Тимофей\Documents\TransformCompressTester

(замени путь на свой)

**Шаг 3: Скомпилируй все файлы**

bash

javac -d bin src/*.java src/crypto/*.java src/gui/*.java

**Шаг 4: Запусти программу**

bash

java -cp bin Main
